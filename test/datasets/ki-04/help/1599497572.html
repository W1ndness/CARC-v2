Tipps zur einfachen Websuche

AltaVista lädt Sie ein, seine Sammlung digitaler Inhalte zu durchsuchen, bestehend aus Milliarden von Webseiten, Datenressourcen und Multimedia-Dateien. Geben Sie einfach eine Abfrage ein und klicken Sie auf die Schaltfläche Suchen.
  • Wenn Sie mehrere Wörter in das Suchfeld eingeben, sucht AltaVista nach Webseiten, die alle Wörter enthalten.
  • Machen Sie Ihre Angaben so spezifisch wie möglich. (Beispiel: Spielergebnisse Bundesliga statt einfach nur Bundesliga)
  • Geben Sie Wörter ein, von denen Sie glauben, dass sie in der gesuchten Webseite enthalten sind. AltaVista indiziert alle Wörter auf jeder Webseite.
  • Wenn Sie nach einer exakten Phrase suchen, setzen Sie sie in Anführungszeichen (Beispiel: "Sein oder nicht sein").
  • Es wird nicht zwischen Groß- und Kleinschreibung unterschieden. Wenn Sie eine bestimmte Groß- oder Kleinschreibung beibehalten möchten, setzen Sie das Wort in Anführungszeichen.
  • Wörter, zwischen denen Satzzeichen stehen, werden so behandelt, als stünden sie in Anführungszeichen. Alle Satzzeichen werden gleich behandelt.
    (Beispiel: Ford.Mustang/Convertible erbringt dieselben Ergebnisse wie "Ford Mustang Convertible".)
  • Wenn Sie Ergebnisse in anderen Sprachen erhalten, können Sie entweder die Webseiten Übersetzen oder Ihre gewünschte Sprache im Suchfeldmenü auswählen.
  • Wenn ein Suchwort einen Akzent enthält, sucht AltaVista nur Wörter mit genau diesem Akzent. Wenn Sie keinen Akzent schreiben, sucht AltaVista Wörter sowohl mit als auch ohne Akzent. So können Sie z. B. nach französischen, deutschen oder spanischen Begriffen suchen, auch wenn Sie nur über eine englische Tastatur verfügen.
Um Ihre Suche weiter einzuschränken, verwenden Sie Erweiterte Web-Suche oder Spezielle Suchbegriffe.

Typen von Webergebnissen
 
Ausführen verschiedener Suchentypen

Die AltaVista-Suchregisterkarten über dem Suchfeld ermöglichen Ihnen eine sofortige Suche in verschiedenen Bereichen: Web, Bilder, Audio, Video, Verzeichnis oder News.

Wenn Sie einen anderen Suchtyp verwenden möchten, klicken Sie einfach auf eine Registerkarte. AltaVista führt die Suche automatisch mit den aktuellen Begriffen im Suchfeld durch. (Leider ist diese Funktion nicht für alle Browser verfügbar.)
 
Features auf Ihrer Websuche-Ergebnisseite

Aktualität der Ergebnisse AltaVista durchsucht täglich das Web, damit unsere Suchergebnisse immer aktuell sind. Wenn eine Seite in unserer Datenbank aktualisiert wurde, bezeichnen wir sie mit "Aktualisiert innerhalb der letzten 24 Stunden" oder "Aktualisiert innerhalb der letzten 48 Stunden"

Übersetzen: Über diesen Link können Sie eine Webseite auf einfache Weise in neun verschiedene Sprachen übersetzen lassen. Wenn Sie eine Seite übersetzen lassen, übersetzt AltaVista automatisch alle Seiten, die mit der Seite über einen Link verknüpft sind.

Verwandte Seiten: Über diesen Link erhalten Sie Webseiten, die der Seite in Ihren Ergebnissen ähneln. (Beispiel: Verwandte Seiten für www.nissan.de erbringt Nissan-Sites in anderen Ländern, sowie Sites für Toyota, Mazda und Mercedes-Benz.)

Mehr Seiten von [dieser Website]: Wenn eine Site mehrere Seiten enthält, die Ihrer Abfrage entsprechen, können Sie auf diesen Link klicken, damit alle Seiten angezeigt werden. Wenn eine zweite Seite in ihrer Relevanz der ersten Seite stark ähnelt, zeigt AltaVista die zweite Seite automatisch eingerückt unter der ersten an.

PDF-Dateien: AltaVista sucht verschiedene Datentypen auf Ihre Abfrage durch. Wenn eine relevante PDF-Datei gefunden wird, erhalten Sie die Meldung "Dateiformat: PDF" und ein Link zum einfachen Download der kostenfreien Adobe Acrobat Reader - Software, die zur Ansicht der PDF-Datei notwendig ist, wird angezeigt.
